Pengaturan Outreach & Database

1. Sinkronisasi Awan (Supabase)

Hubungkan dengan database cloud Supabase Anda agar data tersimpan secara otomatis, terupdate secara real-time, dan bisa disinkronkan di perangkat lain (laptop/HP).

Cara Setup Tabel Supabase (Klik untuk melihat)
  1. Buat project baru di Supabase.
  2. Buka SQL Editor lalu jalankan SQL berikut untuk membuat tabel: 
    CREATE TABLE prospects (
  id TEXT PRIMARY KEY,
  name TEXT,
  niche TEXT,
  service TEXT,
  source TEXT,
  instagram TEXT,
  instagramLink TEXT,
  whatsapp TEXT,
  sentDate TEXT,
  channel TEXT,
  status TEXT,
  followUpDate TEXT,
  notes TEXT,
  updatedAt TEXT
);
  3. Matikan RLS (Row Level Security) untuk tabel prospects agar aplikasi client-side bisa menulis data langsung, atau tambahkan policy public read/write.
  4. Salin Project URL dan Anon Key dari menu Settings -> API ke input di atas.

2. Template Pesan WhatsApp Pitching

Ubah template pesan pitching WhatsApp sesuai keinginan Anda. Karakter di dalam kurung kurawal akan otomatis diganti dengan data prospek.

{name} {niche} {service}

3. Manajemen Database Lokal

Ekspor data Anda ke file CSV atau backup mentah ke file JSON. Anda juga dapat memulihkan (import) backup data Anda di sini atau mengosongkan seluruh database lokal.
